Сажимање низова¶

Сажимање представља избацивање појединих чланова низа и формирање новог низа

Задатак: Иницијализујте један низ од шест целобројних елемената a[6]={1,5,3,2,4,6}. Сажмите низ тако што ћете избацити елементе који имају вредност izbaci=2. На основу циклуса за сажимање написати програм.

Решење:

#include <stdio.h> 
main() 
{ 
	int a[6]={1,5,3,2,4,6},i,j,izbaci,n,k=0;;
	printf("\nUneti niz:\n"); 
	for(i=0;i<6;i++) 
	{ 
    		printf("a[%d]=%d\n",i,a[i]); 
	}
	printf("Unesite elemenat koji izbacujete: ");
	scanf("%d",&izbaci);
	for (int i=0; i<6; i++)
	{
    		if (a[i] != izbaci)
        		a[k++] = a[i];
	}
	n=k;
	printf("\nSazet niz izgleda izgleda :\n");
	for(j=0;j<n;j++) 
	{ 
    		printf("a[%d]=%d\n",j,a[j]); 
	}
}

Задатак: Написати програм којим се уноси n елемената целобројног низа, а затим врши избацивање жељеног елемента. Предвидети да уколико се унесе вредност која се не налази у низу испише поруку „У низу не постоји елемент за избацивање”. Уколико је све у реду исписати сажети низ.

Решење:

#include <stdio.h> 
main() 
{
	int i,a[50],n,izbaci,k=0;
	printf("Unesi koliko elemenata niza unosis ");
	scanf("%d",&n);
	printf ("Unesi %d elemenata niza: \n",n);
	for (i=0; i<n; i++)
	{ 
  		printf ("a[%d]=",i);
		scanf ("%d",&a[i]);
	}
	printf("Unesite elemenat koji izbacujete: ");
	scanf("%d",&izbaci);
	for (int i=0; i<n; i++)
	{
    		if (a[i] != izbaci)
        		a[k++] = a[i];
	}
	if(n==k)
	{
		printf("\nU nizu ne postoji element za izbacivanje");
		printf("\nNiz je ostao nepromenjen\n");
	for(i=0;i<n;i++) 
	{ 
  		printf("a[%d]=%d\n",i,a[i]); 
	}
	}
	else
	{
		n=k;
		printf("\nSazet niz izgleda izgleda :\n");
		for(i=0;i<n;i++) 
		{ 
  			printf("a[%d]=%d\n",i,a[i]); 
		}	
	}
}
